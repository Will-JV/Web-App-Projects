# CSS Cheat Sheet
## Class
```CSS
.name {
    /* Selects all elements with class="name" */
}
```
Class can be reused.

## ID
```CSS
#name {
    /* Selects element with id="name" */
}
```
ID can only be used once.

## Groups of CSS Selectors
```CSS
h1, h2 {
    /* Selected elements are separated with commas */
}
```
